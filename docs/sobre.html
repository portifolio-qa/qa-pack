Sobre o projeto

Conheça o propósito do QA Avancado Pack e os limites desta documentação pública.

Propósito da extensão

O QA Avancado Pack tem como propósito simplificar a preparação do Visual Studio Code para profissionais que trabalham com automação de testes em Java.

A extensão reúne ferramentas úteis para fluxos de QA Automation, incluindo projetos com APIs REST, BDD, Cucumber, Selenium, JUnit e Maven.

Por que foi criada

O projeto foi criado para reduzir o tempo gasto configurando ambientes de trabalho e tornar o início de novos projetos de automação mais consistente.

Com o pack, QAs, SDETs e automadores podem instalar um conjunto inicial de extensões focado em produtividade, organização e suporte ao ecossistema Java.

Aviso

Este repositório contém apenas a documentação pública da extensão. O código-fonte da extensão é mantido em um repositório privado.
