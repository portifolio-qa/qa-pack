# Instalação

Esta página mostra como instalar o **QA Avancado Pack** no Visual Studio Code.

---

## Pré-requisitos

Antes de instalar a extensão, recomenda-se ter:

- Visual Studio Code instalado
- Java instalado
- Maven instalado
- Git instalado
- Acesso à internet para baixar as extensões do Marketplace

---

## Instalação pelo VS Code Marketplace

Acesse a página da extensão no Marketplace:

[📦 Instalar QA Avancado Pack](https://marketplace.visualstudio.com/items?itemName=SEU-PUBLISHER.qa-pack)

Depois clique em:

```text
Install
```

O VS Code será aberto e solicitará a instalação da extensão.

## Instalação diretamente pelo VS Code

No Visual Studio Code:

1. Abra o VS Code.
2. Clique no menu lateral de extensões.
3. Pesquise por:
```text
QA Avancado Pack
```
4. Clique em Install.

## Instalação via linha de comando

Também é possível instalar pelo terminal:

```Bash
code --install-extension SEU-PUBLISHER.qa-pack
```

Exemplo:
```Bash
code --install-extension rosedias.qa-pack
```
## Após a instalação

Após instalar o pacote, o VS Code irá instalar ou sugerir as extensões configuradas no Extension Pack.

Com isso, o ambiente ficará preparado para trabalhar com:

Projetos Java
Projetos Maven
Testes automatizados
Testes de API
BDD com Cucumber/Gherkin
Automação funcional
Produtividade no desenvolvimento

## Verificando a instalação

Para verificar se a extensão foi instalada:

1. Abra o VS Code.
2. Acesse a aba de extensões.
3. Pesquise por:
```text
QA Avancado Pack
```
4. Confirme se o status está como Installed.

## Problemas comuns
A extensão não aparece na busca

Verifique se o nome foi digitado corretamente:
```text
QA Avancado Pack
```
Também é possível acessar diretamente pelo link do Marketplace.

## Algumas extensões não foram instaladas

Em alguns casos, o VS Code pode apenas sugerir a instalação das extensões incluídas no pacote.

Nesse caso, abra a página da extensão e confira a lista de extensões recomendadas.

O comando code não funciona no terminal

No VS Code, pressione:
```text
Ctrl + Shift + P
```
Pesquise por:
```text
Shell Command: Install 'code' command in PATH
```
No Windows, também é possível reinstalar o VS Code marcando a opção para adicionar o comando code ao PATH.
