Instalação

Prepare o Visual Studio Code para trabalhar com automação de testes Java usando o QA Avancado Pack.

Pré-requisitos

Visual Studio Code instalado Java instalado Maven instalado Git instalado Acesso à internet

Instalação pelo Marketplace

Acesse a página do QA Avancado Pack no Visual Studio Code Marketplace e clique em instalar.

Abrir Marketplace

Instalação pelo VS Code

  1. Abra o Visual Studio Code.
  2. Acesse a área de extensões.
  3. Pesquise por QA Avancado Pack.
  4. Clique em instalar e aguarde a conclusão.

Instalação via terminal

Use o comando abaixo quando o comando code estiver disponível no terminal.

code --install-extension rosedias.qa-pack

Problemas comuns

Extensão não aparece na busca

Confirme o nome pesquisado, verifique a conexão com a internet e tente acessar diretamente o link do Marketplace.

Algumas extensões não foram instaladas

Reinicie o VS Code, verifique permissões de rede e repita a instalação do pack.

Comando code não funciona no terminal

No VS Code, habilite o comando de shell pela paleta de comandos e abra um novo terminal.
