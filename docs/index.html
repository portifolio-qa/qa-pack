QA Avancado Pack

Pacote de extensões para QA Automation com Java no Visual Studio Code.

O pack facilita a configuração de ambiente para QAs, SDETs e automadores, reunindo ferramentas úteis para testes automatizados, APIs, BDD e produtividade em projetos Java.
