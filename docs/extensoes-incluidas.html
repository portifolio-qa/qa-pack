Extensões incluídas

O pack reúne extensões para produtividade em QA Automation com Java, organizadas por necessidade de trabalho.

Categorias

Java

Apoia escrita, navegação, execução e manutenção de código Java em projetos de testes.

Maven

Facilita gerenciamento de dependências, ciclos de build e execução de suítes automatizadas.

Testes automatizados

Ajuda no desenvolvimento, leitura e execução de testes unitários, integração e ponta a ponta.

APIs REST

Organiza o fluxo de criação, consumo, validação e documentação de chamadas HTTP.

Cucumber/Gherkin

Melhora a experiência com BDD, escrita de cenários e manutenção de arquivos feature.

Produtividade

Inclui recursos para navegação, leitura, formatação e padronização do trabalho diário.

Organização do ambiente

Contribui para manter o VS Code mais preparado, consistente e rápido para projetos QA.

Resumo das categorias

Categoria Finalidade Exemplos de uso
Java Suporte ao desenvolvimento Java. Editar classes, navegar entre métodos e executar testes.
Maven Gerenciar build e dependências. Executar ciclos de teste, instalar dependências e validar projeto.
Testes automatizados Apoiar criação e execução de testes. Trabalhar com JUnit, Selenium e suítes automatizadas.
APIs REST Testar e validar serviços HTTP. Criar requisições, validar respostas e explorar contratos.
Cucumber/Gherkin Melhorar fluxo BDD. Escrever features, steps e cenários legíveis para negócio.
Produtividade Reduzir atrito no uso diário do editor. Formatar código, organizar arquivos e acelerar navegação.
Organização do ambiente Padronizar o VS Code para automação. Preparar uma base consistente para novos projetos QA.

Os IDs reais das extensões podem ser adicionados posteriormente conforme a composição final do pack.
